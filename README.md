# ResQLink

ResQLink is a community intelligence platform for reporting, prioritizing, and resolving civic, legal, medical, and disaster-related issues through a role-based workflow.

This repository contains:

- A Go backend API and worker
- A Flutter mobile/web client
- Firebase Auth and Firestore integration
- Pub/Sub-based report ingestion
- Gemini/Vertex AI-powered extraction and assistant features
